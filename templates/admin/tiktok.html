📋 Comment obtenir les clés API TikTok
  1. Va sur developers.tiktok.com et connecte-toi avec ton compte TikTok.
  2. Clique sur Manage apps → Connect an app → crée une nouvelle app.
  3. Dans les paramètres de l'app, active les produits Login Kit et Video Kit.
  4. Dans Login Kit → Redirect URL, ajoute exactement :
    http://127.0.0.1:5000/admin/tiktok/callback
  5. Récupère le Client Key et le Client Secret dans l'onglet App info.
  6. Colle-les dans le formulaire ci-dessous et sauvegarde.
  7. Clique Se connecter à TikTok pour autoriser l'accès à tes vidéos.
  8. Une fois connecté, clique Synchroniser pour importer tes covers automatiquement.

⚠️ Ton app TikTok doit être en mode Sandbox ou Live. En sandbox, ajoute ton compte TikTok dans la liste des testeurs autorisés.
